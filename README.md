# Sanss Portfolio

A minimalist, warm, and organic portfolio layout inspired by mid-century modern interior design and curated workspaces, featuring a unique "sketchbook" aesthetic.

## Features
- **Dynamic Sketchbook Aesthetic:** Hand-drawn borders, "pencil" textures, and organic animations.
- **Dedicated Portfolio Page:** Showcasing 5 core projects with interactive hover states.
- **Conference Timeline:** Highlighting teaching and speaking engagements.
- **Responsive Design:** Fully accessible and mobile-friendly.

## Deployment
To push updates to your repository:
```bash
git add .
git commit -m "Update portfolio content"
git push origin main
```

## Setup
Built with React, TypeScript, Tailwind CSS, and Lucide React.
